# Rockbuster_SQL
Analysis of the Rockbuster movie database using SQL
## Background
Rockbuster Stealth LLC is a movie rental company that used to have stores around the world. With stiff competition from digital streaming services, such as Netflix and Amazon Prime, Rockbuster Stealth decided to use its existing movie licenses to launch an online video rental service to stay competitive.
## Objective
Help the business intelligence department answer a series of business questions with data-driven responses that Rockbuster Stealth can use to launch their streaming service.
## Data
This project used data from a relational database containing 17 tables on customers, payments, locations, movies and actors, inventory, rentals, employees, and stores. The full dataset can be found here:
